1. User opens application
   → Sees light theme by default

2. User clicks "🌙 Dark Theme" button
   → Application instantly switches to dark theme
   → Button changes to "☀️ Light Theme"
   → All colors update throughout interface

3. User performs backup/restore operations
   → Theme persists across all screens

4. User wants to switch back
   → Clicks "☀️ Light Theme" button
   → Returns to light theme instantly

Settings to configure: 0 (automatic)
