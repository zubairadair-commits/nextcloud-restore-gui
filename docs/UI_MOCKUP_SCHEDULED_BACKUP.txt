THEME SUPPORT
─────────────

Both Light and Dark themes are supported:

Light Theme:
  • Background:        #f0f0f0
  • Foreground:        #000000
  • Button BG:         #e0e0e0
  • Entry BG:          #ffffff
  • Hint Color:        #666666

Dark Theme:
  • Background:        #1e1e1e
  • Foreground:        #e0e0e0
  • Button BG:         #2d2d2d
  • Entry BG:          #2d2d2d
  • Hint Color:        #999999


SCROLLING
─────────

The entire configuration area is scrollable to accommodate all options.
Mouse wheel support for both Windows and Linux.


USER INTERACTIONS
──────────────────

1. Component Selection:
   - User can check/uncheck optional components (apps, custom_apps)
   - Required components (config, data) cannot be unchecked
   - Default: All components selected

2. Rotation Selection:
   - User selects one radio button
   - Default: "Unlimited (no deletion)"
   - Changes are saved when schedule is created/updated

3. Schedule Creation:
   - Clicking "Create/Update Schedule" validates all inputs
   - If validation passes, schedule is created with selected options
   - Success message shows all configured settings
   - Page remains open for testing with "Test Run" button


ACCESSIBILITY
─────────────

• All interactive elements have tooltips
• Keyboard navigation supported
• Color-blind friendly (icons and text, not just color)
• Clear visual hierarchy with headers and sections
• Disabled state clearly indicated for required components
