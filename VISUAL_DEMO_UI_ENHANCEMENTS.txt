FEATURE 1: TIMEZONE DISPLAY
================================================================================

BEFORE (Old UI):
┌────────────────────────────────────────────────────────────┐
│  Backup Time (HH:MM):                                      │
│  ┌──────────┐                                              │
│  │  02:00   │                                              │
│  └──────────┘                                              │
└────────────────────────────────────────────────────────────┘

Issue: Users don't know what timezone this time represents.

AFTER (Enhanced UI):
┌────────────────────────────────────────────────────────────┐
│  Backup Time (HH:MM):  [UTC+00:00 (UTC)]  ℹ️              │
│                         ^^^^^ NEW! ^^^^^                   │
│  ┌──────────┐                                              │
│  │  02:00   │  ← Hover for format help & timezone info   │
│  └──────────┘                                              │
└────────────────────────────────────────────────────────────┘

Tooltip on timezone label:
┌─────────────────────────────────────────────────────┐
│ Backup times are in your system's local time zone. │
│ The task scheduler will run at this time on your   │
│ local system.                                       │
└─────────────────────────────────────────────────────┘

Tooltip on time entry:
┌─────────────────────────────────────────────────────┐
│ Enter time in 24-hour format (HH:MM)               │
│ Example: 02:00 for 2 AM, 14:30 for 2:30 PM         │
│ Timezone: UTC+00:00 (UTC)                           │
└─────────────────────────────────────────────────────┘

Benefits:
  ✓ Clear indication of timezone
  ✓ No confusion about AM/PM
  ✓ Users know exactly when backup runs
  ✓ Helpful examples and format guidance

KEY IMPROVEMENTS SUMMARY
================================================================================

1. TIMEZONE CLARITY
   • Display format: [UTC+00:00 (UTC)]
   • Position: Next to time picker label
   • Tooltip: Explains system local time
   • Entry tooltip: Provides format examples with timezone

2. CLOUD STORAGE INTEGRATION
   • Auto-detection: OneDrive, Google Drive, Dropbox, iCloud Drive
   • Quick selection: One-click buttons for detected folders
   • Info icon: Comprehensive tooltip about cloud vs. local storage
   • Visual indicators: ☁️ for cloud, 💾 for local

3. STATUS TRANSPARENCY
   • Cloud sync status: Shows provider and "automatic sync enabled"
   • Local-only status: Clear "no cloud sync detected" message
   • Timezone in status: Shows when backup runs with timezone

4. SELF-SERVICE SETUP
   • Step-by-step guide: 3 steps per provider
   • Consistent format: Same structure for all providers
   • Clear expectations: Notes about sync time and behavior

5. GROUNDWORK FOR FUTURE
   • OAuth integration: Cloud detection enables future API connections
   • Direct upload: Status system ready for progress indicators
   • Timezone selection: Display ready for timezone picker addition
