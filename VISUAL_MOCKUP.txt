THEME SWITCHING BEHAVIOR:
========================

Step 1: User sees light theme with "🌙 Dark Theme" button
Step 2: User clicks the theme toggle button
Step 3: Instant switch to dark theme
Step 4: Button text changes to "☀️ Light Theme"
Step 5: All colors update throughout the application
Step 6: User can toggle back anytime with single click

RECURSIVE THEME APPLICATION:
============================

When theme is toggled, the system:
1. Updates root window background
2. Updates header frame and label
3. Updates status label
4. Updates body frame
5. Recursively walks through all child widgets
6. Applies appropriate colors to:
   - Frames (background)
   - Labels (background + foreground)
   - Buttons (background + foreground + active)
   - Entry fields (background + foreground + cursor)
   - Text widgets (background + foreground)
   - Listboxes (background + foreground)
